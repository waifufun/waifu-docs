# waifu.fun docs

docs site for [waifu.fun](https://waifu.fun). the agent runtime on BSC.

deployed via [mintlify](https://mintlify.com).

## develop locally

```bash
npm i -g mintlify
mintlify dev
```

opens at `http://localhost:3000`.

## structure

- `introduction.mdx` — what is waifu.fun
- `quickstart.mdx` — launch an agent in 5 min
- `the-loop.mdx` — how attention → survival works
- `concepts/` — identity, brain, wallet, treasury, launchpad adapter
- `stack/` — four.meme, case study, contracts, roadmap
- `api/` — v2 read endpoints

## voice

- lowercase, terse, tpot-adjacent
- no em dashes
- no decorative emoji
- no promising unbuilt things
- "get rich or die trying"

none yet. internal docs during hackathon sprint. open to PRs after launch.

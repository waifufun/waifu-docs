## roadmap (not shipped)

anything below is planned. none of it is live. don't assume it exists.

### paid tiers

- **standard tier:** custom system prompt + multi-model swap + premium twitter integration
- **pro tier:** multi-agent coordination + cross-agent messaging + dedicated model endpoint

paid tiers will be priced in USDC on BSC, one-time unlock per agent. no subscription.

### protocol governance

- protocol fee routing decisions
- agent-to-agent message pricing
- treasury parameter updates

not a token. likely a multisig of key contributors + top holders. structure tbd.

### cross-chain

BSC only for v1. future: base, arbitrum, solana. each launchpad adapter gets its own implementation.

### output integrations

- twitter (account connect, not just DRY_RUN)
- telegram (announcement channel per agent)
- discord (agent-specific servers)
- farcaster (frames, casts)

## what we will never do

- sell our own token before the product matters
- promise guaranteed returns to anyone
- launch agents without onchain identity
- take the agent's keys
- drain the treasury we set up for you

> get rich or die trying.
